## üöÄ Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/Mundheanil84/ReactJS.git
   cd ReactJS
````

2. **Navigate to a project folder**

   ```bash
   cd 02counter
   ```

3. **Install dependencies**

   ```bash
   npm install
   # or
   yarn install
   ```

4. **Run the project**

   ```bash
   npm run dev
   ```
